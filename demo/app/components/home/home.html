Getting Started

Angular2 building blocks based on Semantic UI

  • Angular 2 angular.io
  • Semantic UI semantic-ui.com
  • GitHub repository https://github.com/vladotesanovic/ngSemantic ( If you like it, give it a star )

Installation

$ npm install ng-semantic --save

In your index.html ( entry file )

<script src="node_modules/ng-semantic/bundles/ng-semantic.min.js"></script>

Semantic css and js ( with jQuery are required )

<link rel="stylesheet" type="text/css" src="/path/to/semantic.min.css"></link>
<script src="/path/to/jquery-2.1.4.min.js"></script>
<script src="/path/to/semantic.min.js"></script>

Import in your project

import {Component} from 'angular2/core';
import { SEMANTIC_COMPONENTS, SEMANTIC_DIRECTIVES } from "ng-semantic";

@Component({
selector: 'demo-cmp',
directives: [SEMANTIC_COMPONENTS, SEMANTIC_DIRECTIVES],
...
