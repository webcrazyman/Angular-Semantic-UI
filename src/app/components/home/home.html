Getting Started

Angular2 building blocks based on Semantic UI

  • Angular 2 angular.io
  • Semantic UI semantic-ui.com
  • GitHub repository https://github.com/vladotesanovic/ngSemantic ( If you like it, give it a star )

Important

  • jQuery is required
  • There is no bundled version yet.

Installation

  • npm install ng-semantic --save
  • Semantic UI must be loaded in your index.html ( entry file )
  • In your index page load:
    • semantic.min.css
    • jQuery
    • semantic.min.js
  • Configure SystemJS loader: Gist link ( config.js )
